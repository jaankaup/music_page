API for https://jaankaup1.ties478.website/Task3/webapi/

concerts

Methods:

GET: lists all concerts in JSON format.

POST: not supported.

PUT: Add a new concert. Query-params: date,name,members. Supported only by admin.

DELETE: not supported.

concerts/{integer}

Methods:

GET: get a specific concert in JSON format.

POST: modify a specific concert. Use JSON format.

PUT: not supported.

DELETE: delete a specific concert.

concerts/{integer}/members

Methods:

GET: get all players and instruments related to a specific concert in JSON format.

POST: modify a specific concert. Use JSON format.

PUT: not supported.

DELETE: delete a specific concert (admin).

members

Methods:

GET: get all available members in JSON format.

POST: not supported.

PUT: Add a new member. query params: name={string}

DELETE: not supported.

members/{integer}

Methods:

GET: get a specific members in JSON format.

POST: modify member (admin): name={string], password={string}.

PUT: now implemented.

DELETE: delete a specific member (admin).

songs

Methods:

GET: get all available songs in JSON format. Query params: name='a song name', userId={integer} are used as a filters.

POST: not supported.

PUT: Add a new song. query params: name={string},concertId={integer}

DELETE: not supported.

songs/{integer}

Methods:

GET: get a specific song in JSON format.

POST: modify a specific song. Only supported by admin.

PUT: not supported.

DELETE: delete a specific song. Only supported by admin.

songs/{integer}/file

Methods:

GET: get the song file in mp3-format.

POST: now implemented.

PUT: add a new mp3 file, maximum file size is 30M (admin).

DELETE: delete a specific song file (admin).

login

Methods:

GET: get the login information.

POST: login. query params: name={string},password={string}.

PUT: not supported.

DELETE: not supported.
